# University-Internship-Job-Board-Web-Application-
The University Internship & Job Board Web Application is a responsive platform designed to help university students easily discover and apply for internships and job opportunities. The system supports multiple user roles, including Admin, Students, and Companies, ensuring secure, role-based access. Key features include posting and managing jobs and internships, search and filter functionalities, application tracking, and notifications for new listings. Professors and company representatives can post opportunities and monitor applications, while students can track their progress efficiently. Built with HTML, CSS, JavaScript, PHP, and MySQL, the application provides a user-friendly interface and a robust backend to streamline the internship and job search process, bridging the gap between academia and industry and improving placement efficiency.
