# Campus Hire – Job Board Web Application
Campus Hire is a responsive Job & Internship Board Web Application designed for university students to search, apply, and manage internship and entry-level job opportunities before graduation.

This system helps students connect with employers and allows administrators to manage job postings efficiently.

🚀 Features
👨‍🎓 Student

Create and manage profile

Upload CV/Resume

Browse internships & job categories

Apply for jobs online

View application status

🛠️ Admin

Add, edit, delete job listings

Manage student accounts

Review applications

Control platform content

🎨 UI Features

Fully responsive design

Modern landing page

Animated sections using ScrollReveal.js

Mobile navigation menu

Clean and user-friendly interface

💻 Technologies Used

HTML5

CSS3

JavaScript

PHP

MySQL

ScrollReveal.js

🎯 Purpose of the Project

This project was developed as part of an Advanced Web Development academic project.
The goal is to provide a simple and effective platform that helps students secure internships and job opportunities early in their academic journey.
